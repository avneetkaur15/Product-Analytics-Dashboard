# 📊 Product Analytics Dashboard

An interactive dashboard developed using **Power BI** and **SQL** to analyze product sales, profitability, discount performance, and country-wise revenue trends.

The dashboard enables users to explore sales performance across different products, countries, discount bands, and years through dynamic visualizations and KPIs.


### 📌 Project Objectives

The primary objective of this project is to:

- Analyze product sales performance.
- Measure profitability across different products and countries.
- Evaluate the impact of discounts on revenue.
- Compare yearly business performance.
- Build an interactive dashboard for business decision-making.

### Dashboard Features

The dashboard includes the following components:

- Monthly Revenue Trend
- Country-wise Revenue Analysis
- Profit Margin KPI
- Net Revenue KPI
- Revenue Distribution by Discount Band
- Product Selection Panel
- Revenue & Profit Matrix

Users can interact with slicers to filter the dashboard dynamically.

### Tools & Technologies

- Power BI
- SQL
- Microsoft Excel (CSV Datasets)
